## Como usar

1. Clonar repositório
``git clone https://github.com/arkanael/ignite-feed-react.git ``

2. Instalar dependências
``yarn`` ou ``npm i``

3. Subir projeto Local
``yarn dev`` ou ``npm run dev``

4. Acessar projeto
``http://localhost:3000/``
